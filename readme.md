stm_rcd
---
This repository contains the firmware for RCD module of PERLA solar car. <br> 
**NOTE**: RCD is a working name, in fact the module contains not only the RCD device but is also responsible for communicating with Type 2 EV charger as well as provides the safety features for the latter.

| PIN   | Name   | Function   |
| :---: | :----: | :--------: |
| PA0 | RCD_PWM | Receives the pwm signal corresponding to the amount of residual current generated by RCD device|
|PA1| TYPE2_LED_RED | Controls red led on type2 connector
|PA2| LED_RED | Controls red led on PCB
|PA3| LED_GREEN | Controls green led on PCB
|PA4| TYPE2_LED_GREEN | Controls green led on type 2 connector
|PA5| L1_SENSOR | Read ADC transoptor value to check for voltage across phase 1
|PA6| L2_SENSOR | Read ADC transoptor value to check for voltage across phase 2
|PA7| L3_SENSOR | Read ADC transoptor value to check for voltage across phase 1
|PA9| USART1_TX | USART hardware communication
|PA10| USART1_RX | USART hardware communication
|PA11| CAN_RX | CAN hardware communication
|PA12| CAN_TX | CAN hardware communication
|PB0| PP | Read ADC value on PP pin to detect max cable current
|PB1| CP | Read PWM value on CP pin to detect max charger current
|PB3| RCD_FAULT | Indicates residual current fault
|PB4| RCD_ERROR | Indicates the RCD device error
|PB6| RCD_TEST | Allows for testing the RCD device
|PB7| START_CHARGING | Connects a resistor to indicate that the car is ready to charge
